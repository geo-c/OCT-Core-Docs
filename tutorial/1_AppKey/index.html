Create you own API key¶

  1. Visit the Open City Toolkit page:

http://giv-oct.uni-muenster.de:8081/

  1. Select Get an API Key. An empty from will open.

  1. Fillin the form with App details

  1. Apply and you will get your API Key

Your key has also been sent to your email address

The URL of the API is: http://giv-oct.uni-muenster.de:8080/api/

You can query the databases with http://giv-oct.uni-muenster.de:8080/api/query/:categoryname:?authorization=YourAPIKey

  1. Use the API Key

When you enter the following get request in you browser you get the JSON responce http://giv-oct.uni-muenster.de:8080/api/query/Traffic?authorization=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJhcHBfbmFtZSI6Ik1heCIsImlhdCI6MTQ3NDk4MzcwMX0.gaJOuK6yPO0s8CaMDHsYCvf0i6bnzDqLR2JO5krdBAs

You can change the query for example to Social. http://giv-oct.uni-muenster.de:8080/api/query/Social?authorization=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9.eyJhcHBfbmFtZSI6Ik1heCIsImlhdCI6MTQ3NDk4MzcwMX0.gaJOuK6yPO0s8CaMDHsYCvf0i6bnzDqLR2JO5krdBAs

As responce you will get an other data collection

Further request you will find hier: http://giv-oct.uni-muenster.de:8080/docs/architecture/rest-api/
